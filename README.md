# MJMusic
##这是一个本地音乐 App Demo具有如下功能
- 1.音乐播放
- 2.音乐切换
  - 2.1 音乐切换模式三种：随机播放、列表播放、单曲循环
- 3.播放列表查看
  - 3.1 播放列表选择想听的歌曲
- 4.播放歌词实时同步动画
- 5.播放页面专辑封面旋转
- 6.背景实现专辑封面蒙版

##待完善部分
- 1.进入后台后，继续播放音乐
- 2.锁屏后，锁屏界面控制音乐的播放，切歌
- 3.耳机拔掉之后，暂停音乐播放

### 使用到的第三方框架有：
	- pod 'Masonry'
	- pod 'MJExtension'
